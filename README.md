# Симулятор артефактов Genshin Impact

Телеграм бот, симулирующий механику артефактов из Genshin Impact. При помощи псевдо рандома эмулирует генерацию подстатов.

# Запуск

Для запуска бота необходимы следующие параметры окружения:

- `bot_token` - токен бота телеграм, который будет отправлять запросы.
- `bot_language` - язык бота. На данный момент поддерживается "ru" и "en".

Запуск бота происходит при помощи `docker-compose`.

`docker compose -f docker-compose.yml --env-file .env up --build`

После билда основной функционал бота будет выполняться в контейнере `backend`.
